Project created to control my robotic pet cat. 
Two seperate code bases 
	-Arduino, for the arduino microcontroller code (sensor input/out, basic collision avoidance, etc)
	-CatBrain, for the Raspberry Pi code (navigation, image recognition, emotion processing)

To make
1) load microcontroller code to Arduino
     -load 'main.ino' in the src folder

2) start xyz? on the raspberry pi
